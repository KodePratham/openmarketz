# OpenMarketz Docs

This folder contains practical operator docs for the AMM-only showcase MVP.

## Docs Index
- `amm-redeployment-checklist.md`: When and how to redeploy AMM safely.
- `amm-env-wiring.md`: Required env updates after AMM redeploy.
- `amm-showcase-runbook.md`: End-to-end demo flow for two wallets.

## Why This Exists
AMM OPEN code support is now on-chain, so deployment and frontend wiring must stay in sync.
